Architecture

In the last meeting, we whiteboarded a solution for Martin based on an event-driven serverless architecture.

Principles

Modern, event driven solution that lays the foundation for further development. The solution should require as little overhead as possible.

Well Architected Framework

The goal was to design a solution specific to this use-case but also flexible enough to evolve over time. Additionally, the solution must follow all of the well architected framework pillars:

  • Operational Excellence
  • Security
  • Reliability. The solution must be fault tolerant and auto-scaling.
  • Performance Efficiency
  • Cost Optimization
